Redis jest otwarto-źródłowym (licencja BSD) magazynem danych przechowywanych w pamięci używanym jako baza danych, cache lub broker wiadomości. Wspiera struktury danych takie jak łańcuchy znaków, hashe, listy, zbiory, zbiory sortowane z zapytaniami promieniowymi, bitmapy, hyperloglogi and indeksy danych przestrzennych z zapytaniami promieniowymi. Redis posiada wbudowaną replikację, obsługę skryptów Lua, usuwanie danych najmniej używanych, transakcje, różne poziomy utrwalania danych na nośniku oraz dostarcza mechanizmy wysokiej dostępności przez Redis Sentinel i automatyczne partycjonowanie za pomocą Redis Cluster.
